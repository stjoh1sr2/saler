Saling, Simplified

Garage Saler is a website for garage sale hosts and potential customers. Users will be able to go on the website to view different garage sale listings. This project will also allow for users to host ads of their own garage sales. When viewing available listings, users can add them to a cart feature, from which the website can generate the most efficient route for the user to travel to each of the sales they chose. We aim to modernize the experience of garage sale hunting.
