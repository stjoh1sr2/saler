Project-Related Resources

Problem-Domain Book

Software Engineering, 10th Edition by Ian Sommerville (ISBN: 9332582696): This book outlines general software development processes and strategies used in software development. This includes strategies regarding initial design stages, development, and maintenance. There is also information about security, application frameworks, and service-oriented software engineering. The latter includes details about RESTful services, which may be relevant to the features of this project.

Project-Related Websites

  • Google Maps API List: This link provides a list of links to different API for Google Maps. There are several options, including the Directions API and Distance Matrix API. Each individual API link contains a developer guide, best practices, and a setup guide for using the API.
  • OpenStreetMap API Wiki: OpenStreetMap is an open-source alternative to Google Maps and another possibility for implementing our routing feature. This wiki page offers links to in-depth information on using OpenStreetMap’s API in development, including a link to a guide for embedding OpenStreetMap into web pages.
  • Adding Search Functionality To Your Website: This web article describes different ways to add search functionality to a website. There is information about Content Management Systems (CMS) and using local Common Gateway Interface (CGI) scripts to implement search functionalities. Additionally, the page lists some search services that can be embedded within a site.
  • Which Database You Should Choose For Web Development?: This Geeks for Geeks article dives into the advantages and disadvantages of using particular database systems in web development. Each database system is described with a short summary, a bullet point list of features, and links to further resources. Eight such database systems are highlighted, including MySQL and MongoDB.
